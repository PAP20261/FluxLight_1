Configuração

🌐 ESP32 Wi-Fi

Alterar em esp32_code.ino

const char* ssid = "O_TEU_WIFI";
const char* password = "A_TUA_PASSWORD";

📡 IP ESP32

Alterar em script.js

http://192.168.1.115

💡 IPs Shelly

Alterar em script.js → SHELLY_IPS[]

🔧 Configuração Hardware

Alterar em esp32_code.ino

#define NUM_LEDS 300
#define LED_PIN 13

🎙️ Controlo por Voz

Usa a Web Speech API do browser — sem chave nem servidor externo.
